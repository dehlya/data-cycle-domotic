# DataCycle Domotic

**Smart-apartment IoT data platform** — Group 14, HES-SO Valais, 2026.

End-to-end pipeline: raw sensor JSON / weather CSV / MySQL apartment dimensions
→ a 3-layer medallion ETL (bronze · silver · gold) on PostgreSQL → ML
predictions in KNIME → Power BI dashboards for tenants and an admin pane for
operators.

## At a glance

```
SMB share (sensor JSON)  ─┐
MySQL (apartment dims)   ─┼─►  Bronze  ──►  Silver  ──►  Gold  ──►  KNIME predictions
sFTP (weather CSV)       ─┘   (raw)        (clean)      (star)      (motion + consumption)
                                                            │
                                                            ├──► Power BI dashboards (RLS per apartment)
                                                            └──► Streamlit admin (status / actions / logs)
```

Everything runs **locally on a single Windows VM**. No cloud dependencies.

## Quickstart for users

1. Visit the project's `/install` page in your browser
2. Fill in the form (DB host, app password, sFTP creds, SMB share, etc.)
3. Download `data-cycle-installer.py` and run it:
   ```
   python data-cycle-installer.py
   ```
4. ~45-60 min later, the admin dashboard auto-launches at <http://localhost:8501>
5. The watcher is registered to start on every login (optional prompt)

For a step-by-step walkthrough see [`INSTALLATION.md`](INSTALLATION.md).

## What's included

| Layer | Tech | Where |
|---|---|---|
| Acquisition | Python (paramiko, aiomysql) | `ingestion/` |
| Storage | Local filesystem + PostgreSQL 17 | `storage/`, `etl/` |
| Orchestration | Custom Python watcher (no Airflow) | `ingestion/fast_flow/watcher.py` |
| ETL | pandas + SQLAlchemy + psycopg2 COPY | `etl/bronze_to_silver/`, `etl/silver_to_gold/` |
| ML | KNIME Analytics Platform 5.x (batch mode) | `ml/knime/` |
| Dashboards | Power BI Desktop + RLS | `bi/power_bi/DataCycleDomotic.pbix` |
| Admin pane | Streamlit | `scripts/admin.py` |
| Installer | Self-contained Python script generated by web wizard | `installer/install_template.py` |
